# HR Analytics – Employee Promotion Analysis
Analyzed 50K+ employee records across multiple departments and performance metrics.

## Project Overview
This project analyzes employee data to identify factors that influence promotion decisions within an organization.

## Problem Statement
Organizations need to understand which factors impact employee promotions in order to improve workforce management and career development strategies.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
HR dataset containing:
- Employee ID
- Department
- Performance rating
- Training hours
- Promotion status

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory data analysis
- Transformed 20+ HR variables including performance scores, training hours, and KPIs.
- Visualization of promotion patterns

## Key Insights
- Employees with **higher performance ratings had a greater probability of promotion**.
- Training participation showed a positive relationship with promotion outcomes.
- Promotion distribution varied significantly across departments.
